Kanhasoft Bulk Invoice Delete (Odoo 17)

This module allows Administrator to delete multiple invoices directly from tree view. It automatically unreconciles payments, resets invoices to draft, and deletes them safely.

Key Features

  • Bulk delete multiple invoices
  • Works for Customer & Vendor invoices
  • Automatically removes reconciliation
  • Resets posted invoices to draft
  • Admin-only access
  • Odoo 17 Compatible

How It Works

  1. Go to Accounting → Customers → Invoices
  2. Switch to List View
  3. Select multiple invoices
  4. Click Force Delete

Warning

This module permanently deletes invoices and journal entries. Use only for testing or controlled environments. Not recommended for production accounting.

Technical Details

  • Model: account.move
  • Method: action_bulk_force_delete()
  • Access: System Administrator only
